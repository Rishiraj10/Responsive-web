LifeAfter

Genre: Survival Game
Initial release date: April 18, 2019 Game
Publisher: NetEase Inc, Hong Kong NetEase Interactive Entertainment Limited, X.D. Global
Developers: Hong Kong NetEase Interactive Entertainment Limited, NetEase Games

Intro:- LifeAfter is the English version of a popular Chinese survival game “The Day After Tomorrow” developed by NetEase. It’s so far the best survival mobile game I’ve ever stumbled upon. I love it so much that I even dedicate my time to write a beginner’s guide for it. I’ve never written any game guides before so this is sort of a big deal….well, at least for me.
